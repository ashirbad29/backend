# README

# Devsnest Backend

* Ruby version, use ruby 2.7 for this (Use RVM)
* Install mysql, preferably version 8 and set up user root and a password
* Clone the repo
* Copy paste sample.env to a .env file
* cd backend/devsnest
* bundle install
* rake db:create
* rake db:migrate
* rails s
